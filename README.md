# 📝 To-Do Web App

A simple and modern To-Do Web Application built using HTML, CSS, and JavaScript. This project helps users manage daily tasks efficiently with a clean interface and smooth interactions.

---

## 🚀 Features

* ➕ Add new tasks
* ✅ Mark tasks as completed
* ❌ Delete tasks
* 🕒 Shows date & time
* 💾 Saves tasks using Local Storage
* 📱 Responsive and user-friendly design

---

## 🛠️ Built With

* HTML
* CSS
* JavaScript

---

## ⚙️ How to Use

1. Enter your task
2. Click **Add**
3. Mark tasks as complete or delete them

---

## 🎯 Purpose

This project was created as part of my internship to practice frontend development and build an interactive web application.

✅ Conclusion

This project demonstrates the development of a simple and interactive To-Do Web App using HTML, CSS, and JavaScript. It helped me understand key concepts like DOM manipulation, event handling, and local storage.
